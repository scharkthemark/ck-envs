# ck-envs

environments for working with cookiecutters

-   **source**: https://github.com/scharkthemark/ck-envs
-   **bug reports**: https://github.com/scharkthemark/ck-envs/issues

## overview

This project contains the files needed to create the necessary conda environments for working with cookiecutter templates. The development environment is used to develop and maintain cookiecutter templates, while the non-development environment is used to create new projects with existing cookiecutter templates.
